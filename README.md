# Worldbuilding Engine

A Claude Code–powered worldbuilding tool for DnD and fictional worlds.
Built around a strict two-layer architecture: a portable engine and per-world config.

Status: early development. Setup script not yet released.

## What this is

Drop raw lore fragments into a vault inbox. Claude transforms them into
well-written, interconnected world entries — invented supporting detail where
needed, contradictions flagged rather than resolved, nothing entering canon
without your approval.

## Project layout

This repository is the engine itself. Individual worlds are generated from it
via `setup.sh` (forthcoming). See `engine/` for the prompts, scripts, and
templates that get distributed.
